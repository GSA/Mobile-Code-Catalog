Census API

Sunlight Foundation's Python client for the US Census Bureau API makes it easy to quickly access the data you need. The ACS, SF1, and SF3 data sets are provided by the API. A Census API key is required.
Link: https://pypi.python.org/pypi/census/0.5
Code: https://github.com/sunlightlabs/census
